# high-end-dsp-rpp-cuda Documentation

This site is published with GitHub Pages and provides direct browser-friendly links to the customer PDF documentation.

## What This Site Contains

- Programming guides for RPP CUDA onboarding
- Compatibility PDFs for project setup and SRAM-related API usage
- FFT API reference PDFs for customer demos and training

## Workflow (High Level)

1. Open a PDF guide from the sections below.
2. Read the matching example workflow in the repository README or module README.
3. Build and run the related example locally if you want to validate the flow end to end.

## Workflow Diagram
